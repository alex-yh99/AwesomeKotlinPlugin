IntelliJ Plugin for [Awesome Kotlin](https://kotlin.link/)

![](/docs/screenshots.gif)

## Plugin Development

#### Prerequisite

- JDK 8u112+
- Intellij IDEA (2016.3+)
- Plugins Enabled: Kotlin

Tech Stack

- Build system: Gradle
- Language: Kotlin
- Frameworks: IntelliJ Platform SDK / Java GUI (Swing)

#### Build / Run

```
# run in sandbox
gradle clean runIde

# build artifact
gradle clean buildPlugin
```
